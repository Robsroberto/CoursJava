## **Exercice 1 – Analyse et traitement d’un nombre (Structures conditionnelles + répétitives)**

**(6 points)**

Écrire un programme Java qui demande à l’utilisateur de saisir **un entier positif N**.

Le programme doit ensuite :

1. Vérifier que la valeur saisie est bien **positive**.

   * Tant que l’utilisateur ne donne pas un entier positif, le programme doit **redemander une saisie**.

2. Une fois la valeur validée, afficher :

   * **Tous les nombres pairs** compris entre 1 et N
   * **Le nombre de valeurs paires trouvées**
   * **La somme de toutes les valeurs impaires** comprises entre 1 et N

Exemple attendu :
Pour `N = 10` → pairs : `2, 4, 6, 8, 10`, et somme des impairs = `1 + 3 + 5 + 7 + 9`.

## **Exercice 2 – Traitement d’un tableau d’entiers (niveau simple)**

**(7 points)**

Écrire un programme Java qui :

1. Demande à l’utilisateur de saisir la **taille d’un tableau** T.
2. Remplit ce tableau avec des valeurs entières saisies par l'utilisateur.
3. Affiche :

   * Le **maximum** du tableau
   * Le **minimum** du tableau
   * La **position** du maximum et du minimum (première occurrence)

## **Exercice 3 – Traitement avancé de tableau (niveau intermédiaire)**

**(7 points)**

On souhaite analyser des données numériques dans un tableau.

Écrire un programme Java qui :

1. Demande la **taille du tableau** T.
2. Remplit le tableau avec des entiers saisis par l’utilisateur.
3. Détermine et affiche :

   * Le **nombre de valeurs positives**, négatives et nulles
   * La **moyenne des valeurs positives**
4. Construit ensuite **un deuxième tableau** T2 contenant **uniquement les valeurs négative** du tableau T.
5. Affiche le tableau T2 (et sa taille réelle).

Remarques :

* Si aucune valeur positive  n’existe, indiquer une moyenne égale à 0.
* Le tableau T2 doit être de **taille adaptée** : pas d’emplacements vides.
